## 👋 About

I design and build **Shadow Chat** — a self-hosted, **end-to-end encrypted** messenger.
From the cryptography and real-time backend to the web and native mobile apps,
everything is built in-house. Privacy isn't a setting here — it's the default.

> The server only ever sees ciphertext. Messages, files and calls can be read
> by their participants and no one else.

<br/>

## ⚔️ How it compares

Most messengers make you trade privacy for convenience — or convenience for privacy.
**Shadow Chat refuses the trade.**

|---|:---:|:---:|:---:|:---:|
| End-to-end encrypted **by default** | ✅ | ❌ ¹ | ✅ | ✅ |
| **No phone number or email** to sign up | ✅ | ❌ | ❌ | ✅ |
| **Self-host** on your own server | ✅ | ❌ | ❌ | ✅ |
| Voice & video calls | ✅ | ✅ | ✅ | ✅ |
| Group chats | ✅ | ✅ | ✅ | ✅ |
| Encrypted files & media | ✅ | 🟡 ¹ | ✅ | ✅ |
