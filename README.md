### About

I design and build **Shadow Chat** — a self-hosted, end-to-end encrypted messenger.
From the cryptography and real-time backend to the mobile app, everything is built
in-house, with privacy as the default rather than an option.

- 🔐 **End-to-end encryption** for messages, files and calls
- 📞 **Voice & video calls** over WebRTC
- 📱 **Progressive Web App + native iOS** from one codebase
- 🛡️ **Self-hosted & private** — no ads, no trackers, no data selling
